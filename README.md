# 豆包去水印工具

去除豆包 AI 生成图片右下角的"豆包AI生成"水印。

## 使用方法

### Windows 用户

下载 `豆包去水印.exe`，双击运行：

- **单张处理**：点击"选择单张图片" → 选中图片 → 自动在同目录生成 `xxx_nologo.png`
- **批量处理**：点击"选择文件夹（批量）" → 选中文件夹 → 自动处理里面所有图片
- **拖拽处理**：直接把图片或文件夹拖到 exe 图标上

### macOS 用户

```bash
delogo "~/Downloads/图片.png"           # 单张
delogo --batch '*.png'                   # 批量当前目录
```

## 输出的文件

在原图同目录生成 `原文件名_nologo.png`，原图不会被修改。
